# MathUnicode (support Ubuntu 64 bit)

```
sudo su
mkdir MathUnicode
cd MathUnicode
```
[click here](https://github.com/engineer-ece/Mathematics/blob/master/app1/Math.jar) Download the file to MathUnicode directory.

```
java -jar Math.jar
```
